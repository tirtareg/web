  1. Wajib melakukan daftar ulang Calon Mahasiswa Baru jalur SNMPTN, Tgl 23 Maret s.d. 06 April
    2021 pada Laman : http://admisi.untirta.ac.id

  2. Lengkapi data pada laman https://kip-kuliah.kemdikbud.go.id/ (Lengkapi dan perbaharui semua data yang dicantumkan di laman tersebut. Contoh: alamat, nomor HP, prestasi, upload foto keluarga, foto rumah (tampak ruang keluarga)

  3. Wajib melakukan daftar ulang Calon mahasiswa Baru KIP Kuliah pada link http://bit.ly/KIPKSNMPTN2021, serta mengupload file berukuran tidak lebih dari 1 MB dengan format nama: No.peserta_Nama_jenis data :

    1. Foto/Scan Kartu Tanda Penduduk (KTP), dengan format nama: No.peserta_Nama_KTP
    2. Foto/Scan Kartu Keluarga (KK), dengan format nama: No.peserta_Nama_KK
    3. Foto/Scan Rekening Listrik (Bulan Terakhir), dengan format nama: No.peserta_Nama_Listrik
    4. Foto/Scan Gaji / Surat Keterangan Penghasilan Ayah, dengan format nama: No.peserta_Nama_Ayah
    5. Foto/Scan Gaji / Surat Keterangan Penghasilan Ibu, dengan format nama: No.peserta_Nama_Ibu
    6. Foto/Scan Gaji / Surat Keterangan Penghasilan Ibu, dengan format nama: No.peserta_Nama_Wali
    7. Foto/Scan Kartu Indonesia Pintar (KIP) / Kartu Keluarga Sejahtera (KKS) / Program Keluarga Harapan (PKH) / Surat DTKS / SKTM, dengan format nama: No.peserta_Nama_KIP/KKS/PKH/DTKS/SKTM.
    8. Foto Rumah Tinggal Tampak Depan, dengan format nama: No.peserta_Nama_Depan
    9. Foto Rumah Tampak Belakang, dengan format nama: No.peserta_Nama_Belakang
    10. Foto Ruang Keluarga, dengan format nama: No.peserta_Nama_Keluarga
    11. Foto Ruang Dapur, dengan format nama: No.peserta_Nama_Dapur
    12. Foto Kamar Mandi, dengan format nama: No.peserta_Nama_Mandi
    13. Rute Perjalanan dari pusat kota/kabupaten ke rumah tinggal, berupa tempat-tempat yang
      dapat dijadikan patokan, dengan format nama: No.peserta_Nama_Rute
    14. Foto/Scan SPPT PBB, dengan format nama: No.peserta_Nama_PBB
    15. Mengisi form surat pernyataan beasiswa bidikmisi bermaterai 10.000, dengan format nama: No.peserta_Nama_Pernyataan. Form ada di lembar kedua.
    16. Fotokopi Sertifikat/Piagam penghargaan atas prestasi akademik/non-akademik yang pernah
      diraih di SMA/SMK/MA/sederajat, dengan format nama: No.peserta_Nama_Prestasi1. (jika
      terdapat lebih dari 1 sertifikat/piagam prestasi silahkan beri angka di akhir)

  4. Bergabung dengan grup Telegram : http://bit.ly/KIPKUntirta2021
